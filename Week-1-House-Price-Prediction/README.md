# 🏠 House Price Prediction

## 📌 Project Overview

The objective of this project is to build a machine learning model that predicts house prices based on different property features. The project demonstrates a complete end-to-end machine learning workflow, including data preprocessing, model training, evaluation and business interpretation of the results.

---

# 🎯 Objectives

- Predict house prices using historical housing data.
- Understand which property features influence house prices the most.
- Compare multiple machine learning models.
- Interpret model results using feature importance analysis.

---

# 📂 Dataset

The project uses a housing dataset containing various features related to residential properties such as:

- Median Income
- Housing Median Age
- Total Rooms
- Total Bedrooms
- Population
- Households
- Latitude
- Longitude

### 5️⃣ Model Evaluation

Models were evaluated using:

- RMSE (Root Mean Squared Error)
- Model Comparison

The most influential features affecting house prices were identified to understand the major drivers of property value.
