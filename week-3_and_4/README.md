# 📈 Sales Forecasting & Demand Intelligence System

An end-to-end retail analytics project that combines **sales forecasting**, **anomaly detection**, **product demand segmentation**, and an **interactive Streamlit dashboard** to support business decision-making.

---

## 📌 Project Objectives

- Forecast future retail sales
- Compare multiple forecasting models
- Detect unusual sales behaviour
- Segment products based on demand
- Generate executive business insights
- Build an interactive dashboard for decision makers
